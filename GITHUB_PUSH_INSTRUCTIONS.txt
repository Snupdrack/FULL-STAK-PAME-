🚀 PASOS FINALES PARA GITHUB (5 MINUTOS)

### PASO 1: Crear Repositorio en GitHub

1. Ve a: https://github.com/new
2. Ingresa con tu cuenta (crear si no tienes)
3. Llena los datos:
   - Repository name: "coner"
   - Description: "CONER - Consultores Especializados en Retiros"
   - Visibility: Public (o Private)
   - NO marques README (ya tienes)
4. Click en "Create repository"

### PASO 2: Copiar la URL HTTPS

Verás una pantalla con comandos. COPIA esta URL:
```
https://github.com/tu-usuario/coner.git
```

### PASO 3: Generar Personal Access Token

1. Ve a: https://github.com/settings/tokens/new
2. Nombre: "CONER Deploy Token"
3. Permisos: Marca ✓ "repo"
4. Expiration: "90 days"
5. Click "Generate token"
6. **COPIA EL TOKEN** (solo aparece una vez)

### PASO 4: Ejecutar en Terminal

En tu terminal, copia y ejecuta estos comandos:

```bash
cd "/c/Users/snupd/OneDrive/Desktop/APP CONER/INTERFAS-CONER--main/INTERFAS-CONER--main"

# Agregar remote origin (reemplaza URL_DEL_REPO)
git remote add origin https://github.com/tu-usuario/coner.git

# Cambiar rama a main
git branch -M main

# Hacer push al repositorio
git push -u origin main
```

### PASO 5: Ingresar Credenciales

Cuando Git te pida credenciales:
```
Username: tu-usuario-github
Password: [Pega aquí el token que copiaste]
```

### PASO 6: Verificar

Después del push, verás algo como:
```
Branch 'main' set up to track remote branch 'main' from 'origin'.
```

¡Abre tu navegador y ve a: https://github.com/tu-usuario/coner

¡Tu proyecto está en GitHub! 🎉
