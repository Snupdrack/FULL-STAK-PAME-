🚀 CÓMO INICIAR EN 3 PASOS

OPCIÓN 1: Manual (Recomendado para Desarrollo)

  Terminal 1 (Backend):
  $ cd backend
  $ python -m uvicorn server:app --reload --port 8000

  Terminal 2 (Frontend):
  $ cd frontend
  $ npm install
  $ npm start

  Accede a: http://localhost:3000

───────────────────────────────────────────────────────────────────────────────

OPCIÓN 2: Docker Compose (Local Completo)

  $ docker-compose up -d

  Espera 30 segundos y abre:
  $ docker-compose logs -f

  Accede a: http://localhost:3000

───────────────────────────────────────────────────────────────────────────────

OPCIÓN 3: PM2 (Process Manager)

  $ npm install -g pm2
  $ pm2 start ecosystem.config.js
  $ pm2 monit

  Accede a: http://localhost:3000
